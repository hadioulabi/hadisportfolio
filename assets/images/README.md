# Bildablage

Hier nur freigegebene oder anonymisierte Assets ablegen.

Empfohlene Struktur:

```text
assets/images/cases/
  ki/
  erp/
  export/
  messe/
  odoo/
```

Vor oeffentlicher Nutzung pruefen:

- keine Kundennamen
- keine Rechnungsdaten
- keine Sendungsnummern
- keine Mitarbeiterdaten
- keine internen URLs, Tokens oder Zugangsdaten
- keine vertraulichen Systemdetails
