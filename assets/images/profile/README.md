# Profilfoto

Lege dein Profilfoto hier ab:

```text
assets/images/profile/hadi.jpg
```

Die Startseite versucht automatisch, diese Datei zu laden. Wenn sie noch nicht existiert, bleibt der Initialen-Platzhalter sichtbar.

Empfehlung:

- Dateiname: `hadi.jpg`
- Quer- oder Hochformat funktioniert, wird automatisch zugeschnitten
- gute Auflösung, aber nicht riesig; etwa 1200 px Kantenlänge reicht
- keine sensiblen privaten Informationen im Hintergrund
